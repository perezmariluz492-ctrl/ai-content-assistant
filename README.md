# AI Content Assistant

AI Content Assistant es un proyecto de código abierto que ayuda a creadores de contenido a generar ideas, títulos, hashtags y prompts utilizando inteligencia artificial.

## Características

- Generador de títulos
- Generador de hashtags
- Generador de prompts para imágenes
- Ideas para Instagram, TikTok y X

## Objetivo

Crear una herramienta gratuita y de código abierto que cualquier persona pueda usar, mejorar y compartir.

## Estado del proyecto

Actualmente en desarrollo.
